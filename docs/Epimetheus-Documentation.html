What is Epimetheus?

Epimetheus is a desktop AI copilot that listens to your meetings and interviews in real-time, transcribes everything, and surfaces AI-powered suggestions and summaries — all running locally with your own API keys.

It is a fork of Natively (the open-source Cluely alternative), renamed and customized for personal and professional use. No subscriptions. No data sent to third-party servers beyond your chosen API providers.

How it Works

Every session follows this pipeline:

🎙 Mic / System Audio
→
ElevenLabs Scribe v2 RT
→
Transcript
→
Local RAG Context
→
Groq Llama 3.3 70B
→
Suggestions on Screen
  • Mic — captures your voice
  • System Audio — captures the other party (Zoom, Meet, etc.)
  • ElevenLabs Scribe v2 Realtime — WebSocket stream, partial transcripts every ~150ms
  • RAG — pulls relevant context from past meetings / uploaded docs
  • Groq — generates real-time coaching, answers, summaries

Environment (.env)

Located at /Users/puttstrife/epimetheus/.env

# ── Epimetheus Environment Config ──────────────────

# ElevenLabs (STT — Scribe v2 Realtime)
ELEVENLABS_API_KEY=your_key_here

# Groq (LLM — Llama 3.3 70B, free)
GROQ_API_KEY=your_key_here

# Optional: Anthropic Claude (better reasoning)
CLAUDE_API_KEY=your_key_here

# Optional: leave blank
OPENAI_API_KEY=
GEMINI_API_KEY=
USE_OLLAMA=false
DEFAULT_MODEL=claude-sonnet-4-6
Keys in .env are auto-loaded on startup. The app will auto-set ElevenLabs as the STT provider and Groq as the default LLM if no other configuration exists.

API Keys Guide

ElevenLabs

  1. Go to elevenlabs.io → Log in
  2. Profile → API Keys → Create API Key
  3. Name it Epimetheus
  4. Enable: Speech to Text, Models, Voices (Read)
  5. Paste into .env as ELEVENLABS_API_KEY

Groq

  1. Go to console.groq.com → Sign up (free)
  2. API Keys → Create API Key
  3. Name it Epimetheus
  4. Paste into .env as GROQ_API_KEY
Groq is completely free with generous rate limits. No billing setup required.
